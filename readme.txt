Backend-приложение для автомата напитков.

Приложение работает локально, для запуска небходимо:
1.Копировать папку App_Data из директории: ...JuiceAutomatMachine\JuiceAutomatMachine\
в директорию: ...JuiceAutomatMachine\JuiceAutomatMachine\bin\Debug\net6.0\

в папке App_Data находится файл с базой данных MSDBData.mdf и файл журнала базы данных MSDBLog.ldf
 
2.Открыть проект в Visual Studio и запустить в режиме отладки.

Приложение должно запуститься автоматически в отдельном экземпляре браузера Google Chrome.
При запуске откроется окно приветствия по адресу: https://localhost:44321/home в браузере.

Далее необходимо запустить Frontend-приложение. (https://github.com/Oleg0392/Juice-machine-test-task.git)
